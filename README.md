# OpenYida Skills Market 🛒

欢迎来到 ** OpenYida Skills Market** - 一个社区共建的 AI Agent Skills 市场！

这里收集了各种第三方开发的 Agent Skills，你可以自由下载使用，放到你的项目维度中，开发个性化的宜搭应用。

## 🚀 快速开始

### 方式一：下载单个 Skill

1. 浏览 `skills/` 目录，找到你需要的 Skill
2. 下载对应的 Skill 文件夹
3. 复制到你项目的 skills 目录中，例如：
   - Claude: `.claude/skills/`
   - Aone Copilot: `.aone_copilot/skills/`
   - 其他 Agent: 参考对应 Agent 的文档

### 方式二：下载 skill.zip

1. 在 GitHub 仓库页面点击 "releases" -> "Download ZIP"
2. 解压后选择需要的 Skill
3. 复制到你项目的 skills 目录

## 🔍 可用 Skills
