# Example Skill

> 这是一个示例 Skill 模板，用于演示如何创建标准的 Skill。

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/openyida/skills-market.git

# 复制到你的 skills 目录
cp -r skills-market/skills/example-skill ~/.claude/skills/
```

### 使用

在你的项目中启用这个 Skill 后，Agent 将自动获得示例能力。

## 功能列表

| 功能 | 描述 |
|------|------|
| 模板演示 | 展示 Skill 的标准结构 |

## 常见问题

### Q: 这个 Skill 有什么实际用途？

A: 这是一个模板，主要用于演示如何创建 Skill。你可以基于这个模板创建自己的 Skill。

### Q: 如何创建自己的 Skill？

A: 复制 `example-skill` 文件夹，修改 `SKILL.md` 文件即可。
